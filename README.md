# Курсовая работа: «Резервное копирование» (на примере CATAAS → Яндекс.Диск)

## Описание
Программа делает резервное копирование изображения кота с текстом из сервиса **cataas.com** в облачное хранилище **Яндекс.Диск**.

Сценарий использования: хотим показать друзьям картинку из интернета, но сайт может быть недоступен. Программа сохраняет копию картинки на Яндекс.Диске, чтобы она была доступна позже.

---

## Что делает программа
1. Запрашивает у пользователя:
    - текст для картинки (этот же текст используется как имя файла на Яндекс.Диске);
    - OAuth-токен Яндекс.Диска (полученный в Полигоне).
2. Формирует URL картинки кота с текстом через API cataas.com.
3. Создает папку на Яндекс.Диске с названием группы в Нетологии: **`spd-140`**.
4. Загружает картинку на Яндекс.Диск **по URL** (без сохранения файла на локальный диск).
5. Получает метаданные загруженного файла (в том числе `size`) и сохраняет отчёт в `result.json`.

---

## Требования
- Python 3.10+ (работает и на более новых версиях)
- Установленные зависимости из `requirements.txt`
